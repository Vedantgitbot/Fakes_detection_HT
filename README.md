# Fakes_detection_HT
Fake Image DetectorA small, honest proof-of-concept that combines classical image-forensics with an LLM reasoning layer to flag possibly AI-generated images.Disclaimer: This is a portfolio/learning project, not a production-validated detection tool.⚙️ How It WorksFeature Extraction: Extracts four numeric forensic features from an uploaded image:FFT high-frequency energy ratioError Level Analysis (ELA) variance and meanNoise-residual "perfection" score (variance-of-local-variances)EXIF metadata presence (kept for completeness, not used by the model)Classification: Classifies the image using two simple scikit-learn models (Logistic Regression & shallow Random Forest) evaluated with Leave-One-Out Cross-Validation (LOOCV).LLM Explanation: Passes the numeric features and dataset baselines (never the raw image) to a Groq-hosted Llama 3.3 70B to generate a plain-language explanation.Dashboard: Users upload an image to a Streamlit dashboard to view classifier scores, raw features, and the AI-generated explanation.📊 Dataset UsedThe model is trained and evaluated using subset samples from the following public dataset:Dataset: 140k Real and Fake Faces (Kaggle)🚀 Quick Start1. 🧠 Honest Design Choices & LimitationsWhat's Unique (The Honest Approach)No Image Hallucinations: The LLM is strictly forbidden from "seeing" or describing the image. It only reasons over numeric features and baselines. Any visual description (e.g., "the skin looks off") is a hardcoded failure condition.Bug Hunting: Identified and resolved a critical bug where bicubic image resizing acted as an unintended low-pass filter, reversing the FFT high-frequency signal.Built to Say "I Don't Know": The UI flags warnings when classifiers disagree by $>30$ points, offers a "Flag for Human Review" option, and uses LOOCV to avoid over-optimistic performance metrics on small sample sizes.Core LimitationsSmall Sample Baseline: The current accuracy is a localized directional estimate on the active training subset, not a generalized benchmark.ELA Confounding: ELA features remain highly sensitive to compression history (JPEG vs. WebP, quality levels) rather than pure authenticity.Local Scope: Built strictly as a local, single-user dashboard with no external API, auth, or production hardening.🛠️ StackPython • Pillow • NumPy • pandas • scikit-learn • joblib • Streamlit • Groq API (Llama 3.3 70B)
